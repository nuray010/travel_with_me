# 🌍 Travel With Me

### Discover. Plan. Explore. ✈️

Travel With Me is a web application designed for travelers who want to organize their trips, explore destinations, and keep all travel information in one place.

Whether you're planning a weekend getaway or your next big adventure, Travel With Me helps make the journey easier.

---

## ✨ Features

🌎 Explore travel destinations

📍 Create and manage travel plans

📝 Store trip information

👤 User-friendly experience

🐳 Dockerized setup for easy deployment

🔄 Database migrations with Alembic

⚡ Fast and modern backend architecture

---

## 🛠️ Tech Stack

**Backend**

* Python
* FastAPI
* SQLAlchemy
* Alembic

**Database**

* PostgreSQL

**Infrastructure**

* Docker
* Docker Compose
* Nginx

---

## 🚀 Getting Started

### Clone the repository

```bash
git clone https://github.com/nuray010/travel_with_me.git
cd travel_with_me
```

### Run with Docker

```bash
docker-compose up --build
```

The application will be available at:

```text
http://localhost
```
